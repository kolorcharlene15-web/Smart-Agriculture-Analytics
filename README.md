


# 🌾 Smart Agriculture Analytics : From Sensor to Harvest

## 🌍 Contexte et Objectif
Ce projet explore l'impact de divers facteurs environnementaux et agronomiques sur le rendement des cultures. En exploitant des données de capteurs, l'objectif est d'aider à la prise de décision pour optimiser les pratiques agricoles, en analysant des variables telles que l'humidité, la pluviométrie et l'utilisation de pesticides, dans le but d'assurer une agriculture durable.

## 🛠️ Outils et Méthodologie
- **Exploration des Données :** Analyse et modélisation à partir des données de la ferme.
- **Variables Clés :** Exploitation de données incluant le type de culture, l'indice NDVI, l'humidité du sol, la pluviométrie, et le type d'irrigation.
- **Visualisation :** Création de tableaux de bord interactifs permettant de visualiser la répartition du rendement par région et selon les types de fertilisants utilisés.
